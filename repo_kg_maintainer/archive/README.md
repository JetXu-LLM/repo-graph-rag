# Archived Research

This directory marks research code that is intentionally excluded from the
public support surface.

Archived components are kept in git history for provenance, but are no longer
shipped as part of the supported runtime because they are broken, exploratory,
or environment-coupled in ways that would weaken the public repository.

Current archived items:

- `repo_doc_maintainer.py`
- `code_analyzer-ckm0-j4kq29y4nc.py`

These components remain valuable as historical context, but they are not:

- part of the default install path
- part of the default test or CI surface
- part of the recommended quickstart
