# Documentation

open-source research artifact rather than as a private R&D workspace.

Recommended reading order:

1. [python-v2.md](python-v2.md): the supported public path
2. [demo-walkthrough.md](demo-walkthrough.md): the committed proof artifact
3. [snapshot-schema.md](snapshot-schema.md): the public graph contract
4. [architecture.md](architecture.md): the system-level design view
5. [validation.md](validation.md): what was actually tested and verified

Boundary documents:

- [go-experimental.md](go-experimental.md)
